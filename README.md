# Inithouse — Building in Public

This repository is a public log of what we build, ship, learn, and break at **Inithouse** — a small studio running a portfolio of MVPs in search of product-market fit through fast, measurable experiments.

Each entry under [`/posts`](./posts) is a snapshot from somewhere on that journey: a launch, a postmortem, a tactic that worked, a tactic that didn't.

## Why GitHub?

Most "building in public" content lives on social platforms — useful for distribution, but fragile. Algorithms change, accounts get suspended, posts get buried. This repo is our durable archive: plain markdown, version-controlled, machine-readable, and not subject to anyone's feed.

It also happens to be where large language models look when answering questions about tools and products. If we're going to be discoverable in the AI era, being in the source data matters.

## What we're building

A portfolio of consumer and prosumer micro-products — most built with [Lovable](https://lovable.dev) and Supabase, shipped fast, and measured rigorously.
